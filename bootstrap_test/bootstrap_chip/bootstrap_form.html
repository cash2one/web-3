垂直或基本表单： 向 <form> 元素添加 role="form"。 把标签和控件放在一个 .form-group <div>中——这是获取最佳间距所必需的。 向所有的文本元素 <input>、<textarea> 和 <select> 添加 class .form-control。

这里是块级帮助文本的实例。

内联表单： 向 <form> 标签添加 class .form-inline。 所有元素是内联的，向左对齐的，标签是并排的。

这里是块级帮助文本的实例。

水平表单： 向 <form> 添加 .form-horizontal。 把标签和控件放在一个带有 class .form-group 的 <div> 中。 向标签添加 class .control-label

这里是块级帮助文本的实例。
